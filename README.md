# 멋쟁이 사자처럼 프론트엔드 첫 만남 JS 입니다!

```
git clone https://github.com/koo-rogie/js-start.git
```

## 해당 문서는 아래와 같이 작성되어 있습니다

```pgsql
js-start
├─ .vscode                => # 선생님 파일 수정 금지
├─ PPT
├─ boj                    => # 백준 문제 그 외 다수
├─ workspace
│  ├─ ch01                => # JS 기초
│  ├─ ch02                => # 함수
│  ├─ ch03                => # prototype
│  ├─ ch04                => # 캐시 및 클로저, 클래스
│  ├─ ch05                => # DOM 객체
│  ├─ ch06                => # TypeScript
│  ├─ css
│  ├─ img
│  ├─ js
│  └─ index.html
├─ .gitignore
├─ README-save.md         => # 선생님 리드밋 파일 분리
├─ README.md              => # 내 폴더의 리드밋
├─ package-lock.json
└─ package.json           => # 라이브 서버 구동을 위한 서버 설치
```
